MigSim

This is the project webpage for MigSim, a tool for simulating cloud database database migration. MigSim consumes KDM and SMM models which can be obtained using DMLModeller. It is an extension of the CloudSim discrete-event simulation framework.

Getting Started

MigSim is packaged as JAR file which can be downloaded from here, the source code is also on GitHub. Java 8 or higher must be installed. It has been tested on Ubuntu, but should run equally well on Windows.
